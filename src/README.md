# 前言

2019 年的一个下午，我从图书馆借出 Hartley & Zisserman 的 *Multiple View Geometry in Computer Vision*。这本书被 CV 领域奉为圣经，封面上印着牛津大学出版社的标志。我翻开第一章，读到第三页就开始头疼。

不是因为数学太难。是因为我不知道**这些数学在解决什么问题**。

等到我在项目里被相机标定、深度估计、3D 重建轮番揍过一遍之后，再回去翻 H&Z，才突然意识到它是一本极好的书——但它的好，需要你先"受够了罪"才能体会到。

这本书想做的，就是让你少受点罪。

## 写给谁

这本书写给对 3D 视觉有兴趣、但被传统教材的门槛挡在外面的人。你可能是：

- 刚入门 CV 的学生，想理解"深度估计""SfM""NeRF"到底是什么
- 做机器人/自动驾驶/AR/VR 的工程师，需要快速搞明白传感器选型、坐标变换、多视图几何
- 转行进入 3D 视觉的开发者，被 H&Z 的数学密度劝退过一次

**你不需要读完 H&Z 才能看这本书。** 每一个概念都从直观场景出发——先告诉你"这解决什么问题"，再推导原理，最后落到代码。

## 为什么写

三个原因。

**第一，面向初学者的 3D 视觉书太少。** H&Z 是写给研究者看的——它追求严谨、完备，但不太关心"一个新手第一次接触对极几何时脑子里在想什么"。Szeliski 的《Computer Vision: Algorithms and Applications》覆盖面广，但 1200 页的体量让人望而生畏。3D 视觉需要一个温和的入口，这本书就是。

**第二，AI 发展太快，缺少"拆解知识"的书。** NeRF 出来三年，3D Gaussian Splatting 横空出世；基础模型一年一换代，端到端方法正在挑战传统的几何路线。论文能告诉你在某个 benchmark 上提升了几个点，但很少告诉你"它为什么有效""它跟前人的区别在哪""你为什么需要学它"。这本书试图做这种拆解——把前沿工作的核心思想提炼出来，放进统一的框架里。

**第三，我希望中文社区有一个好的前沿论文入口。** 很多高质量的 3D 视觉教程是英文的。这本书不打算翻译它们——我要做的是把它们消化之后，用我自己的语言重新组织，融入我在实际项目里踩过的坑。每一章末尾的"关键论文清单"，就是我帮你选的起点。

## 怎么读

这本书分两层：

**基础篇（01-07）** 是必修的。它从"相机怎么成像"讲到"用什么传感器获取深度"，搭建了 3D 视觉的概念脚手架。如果你只有一周时间，把基础篇读完——它够你用很久。

**主题模块（A/B/C）** 是选修的。每个模块深入一个具体子领域，包含直观理解、原理解析、部署实战三阶。你可以根据自己的项目需要跳着读。

**附录** 里收录了一些值得读的延伸文章，包括一篇关于"3D 几何会不会被端到端方法替代"的讨论——也许是你读完基础篇之后最想找人聊的话题。
