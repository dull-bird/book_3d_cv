# 第 6 章 优化基础：从噪声数据中得到最好解释

> [!NOTE]
> **预计阅读时间**：45 分钟 · **前置知识**：第 1-5 章
>
> 3D 视觉里的数据几乎都有噪声：角点不准、匹配有错、深度有空洞、传感器会漂。本章讲优化的基本语言：**参数、观测、残差、代价函数、最小二乘、RANSAC、Bundle Adjustment**。

---

## 6.1 本章目标

前几章出现了很多“求”：

- 求单应矩阵 $H$
- 求基础矩阵 $F$
- 求相机外参 $R,t$
- 求三维点 $X$
- 求内参 $K$ 和畸变参数

真实数据中，这些方程几乎不会被精确满足。优化要解决的问题是：

> 找一组参数，让模型预测尽可能接近观测数据。

统一写法：

$$\min_\theta \sum_i \rho(\|r_i(\theta)\|^2)$$

其中：

- $\theta$：要估计的参数，比如 $H,F,R,t,X,K$
- $r_i$：第 $i$ 个残差，表示预测和观测的差
- $\rho$：损失函数，普通最小二乘时 $\rho(s)=s$，鲁棒优化时会压低 outlier 的影响

这条式子看起来抽象，但可以用一句话翻译：

```text
不断调整参数 theta，让所有观测的预测误差 r 尽量小。
```

在 3D 视觉里，最常见的预测就是“把 3D 点投影回图像”。如果预测像素和真实观测像素差很多，说明相机位姿、三维点、内参或匹配关系里至少有一项不对。

---

## 6.2 残差：优化问题的核心

优化不是“让公式成立”，而是“定义什么叫错”。

常见残差：

| 问题 | 参数 | 残差 |
|------|------|------|
| 拟合直线 | 直线参数 | 点到线的距离 |
| 单应矩阵 | $H$ | 投影点和观测点的像素误差 |
| PnP | $R,t$ | 3D 点重投影到图像后的误差 |
| 三角测量 | 3D 点 $X$ | 两张图的重投影误差 |
| BA | 相机 + 3D 点 | 所有观测的重投影误差 |

例如重投影误差：

$$r_i = x_i - \pi(K,R,t,X_i)$$

其中 $\pi(\cdot)$ 是投影函数。它的单位是 pixel，含义直接：预测像素和观测像素差了多少。

> [!TIP]
> 代数误差容易算，但不一定有物理意义；几何误差更难优化，但单位清楚。3D 视觉里最终通常会回到重投影误差。

残差的单位会影响阈值怎么设。比如 RANSAC 里 `threshold=1.0` 如果对应的是像素误差，大致表示“预测点离观测点 1 个像素以内算 inlier”；如果残差是归一化坐标或代数误差，这个数字就没有同样直观的含义。

---

## 6.3 线性最小二乘

最简单的优化是线性最小二乘：

$$\min_x \|Ax-b\|^2$$

正规方程：

$$A^TAx=A^Tb$$

代码：

```python
import numpy as np

# Fit y = ax + b
x = np.linspace(0, 10, 20)
y = 2.0 * x + 1.0 + np.random.normal(scale=0.5, size=len(x))

A = np.column_stack([x, np.ones_like(x)])
theta, *_ = np.linalg.lstsq(A, y, rcond=None)
a, b = theta
print(a, b)
```

DLT 求 $H$、8 点法求 $F$ 都是类似思想：把几何约束改写成线性方程，先求一个初值。

## 6.4 非线性最小二乘

投影里有除法，旋转也不是线性空间，所以很多问题最终是非线性的：

$$\min_\theta \sum_i \|r_i(\theta)\|^2$$

常用方法：

| 方法 | 直觉 |
|------|------|
| Gauss-Newton | 用局部线性近似迭代更新 |
| Levenberg-Marquardt | 在 Gauss-Newton 和梯度下降之间折中 |
| Gradient Descent | 沿梯度方向慢慢走 |

非线性优化需要初值。DLT、8 点法、PnP 线性/解析解，经常就是为了给后续非线性优化提供初值。

这里的“初值”不是形式要求，而是实际会影响结果。非线性优化每次只相信当前位置附近的局部近似；如果一开始离正确答案太远，它可能走到错误局部最优，甚至直接发散。

> [!TIP]
> 工程上常见链路是：先用 RANSAC 找一批可信 inlier，再用线性方法求初值，最后用非线性最小二乘精修。

---

## 6.5 RANSAC：先把错匹配挡在门外

真实匹配点里常有 outlier。如果直接最小二乘，一个错误点可能把结果拉歪。RANSAC 的做法是：

1. 随机抽最小样本。
2. 拟合一个模型。
3. 用这个模型测试所有点，数 inlier。
4. 重复多次。
5. 选择 inlier 最多的模型。
6. 用所有 inlier 重新估计模型。

例子：

| 模型 | 最小样本 |
|------|---------|
| 单应矩阵 $H$ | 4 对点 |
| 基础矩阵 $F$ | 7 或 8 对点 |
| PnP | 3 或 4 对 3D-2D |

> [!TIP]
> RANSAC 不是让模型更精细，而是让估计不被明显错误的数据污染。它通常和后续最小二乘 refinement 配合使用。

OpenCV 中你已经见过：

```python
H, mask = cv2.findHomography(src, dst, cv2.RANSAC, 3.0)
F, mask = cv2.findFundamentalMat(pts1, pts2, cv2.FM_RANSAC, 1.0)
E, mask = cv2.findEssentialMat(pts1, pts2, K, cv2.RANSAC, 0.999, 1.0)
success, rvec, tvec, inliers = cv2.solvePnPRansac(obj, img, K, dist)
```

## 6.6 鲁棒损失：不是所有残差都同等可信

RANSAC 适合先粗筛 outlier。进入非线性优化后，还可以用鲁棒损失降低大残差权重。

普通 L2：

$$\rho(s)=s$$

对大残差非常敏感。常见鲁棒损失：

| 损失 | 特点 |
|------|------|
| Huber | 小残差像 L2，大残差像 L1 |
| Cauchy | 强烈压制大残差 |
| Tukey | 超过阈值后几乎忽略 |

> [!TIP]
> 鲁棒损失不是替代好数据。它只是承认现实：有些观测可能错了，不应该让它们主导整个解。

## 6.7 Bundle Adjustment：把所有相机和点一起优化

Bundle Adjustment（BA）是 SfM/SLAM/3DGS 中最重要的优化问题之一。

给定：

- 多个相机参数 $\{K_j,R_j,t_j\}$
- 多个 3D 点 $\{X_i\}$
- 图像观测 $x_{ij}$：第 $i$ 个点在第 $j$ 个相机中的像素

优化：

$$\min_{\{R_j,t_j,X_i\}} \sum_{(i,j)} \|x_{ij}-\pi(K_j,R_j,t_j,X_i)\|^2$$

BA 的意义：

- 三角测量得到的点可以 refine。
- PnP 得到的相机位姿可以 refine。
- 所有帧共享约束，误差能全局传播。

BA 的难点：

- 参数很多。
- 矩阵稀疏但规模大。
- 旋转需要特殊参数化。
- 容易受初值和 outlier 影响。

常见工具：

| 工具 | 用途 |
|------|------|
| Ceres Solver | 通用非线性最小二乘，BA 常用 |
| g2o | 图优化，SLAM 常见 |
| GTSAM | 因子图优化 |
| scipy.optimize | 小规模 Python 原型 |

## 6.8 优化在前几章中的位置

| 章节 | 线性初值 | 鲁棒/非线性优化 |
|------|----------|----------------|
| 第 3 章单应矩阵 | DLT 求 $H$ | RANSAC + 重投影误差 refine |
| 第 3 章标定 | 多个 $H$ 线性求 $K$ | 联合优化 $K$、畸变、外参 |

| 第 4 章三角测量 | 线性三角化 | 最小化重投影误差 |
| 第 2 章 PnP | P3P/EPnP | LM refine 位姿 |
| 3DGS | SfM 初值 | 联合优化高斯和相机参数 |

> [!TIP]
> 很多 3D 视觉系统都是两阶段：先用线性/几何方法快速拿到初值，再用非线性优化把误差压低。

## 6.9 常见陷阱

| 陷阱 | 表现 | 对策 |
|------|------|------|
| 没有好初值 | 优化发散或局部最优 | 先用 DLT/PnP/三角测量初始化 |
| outlier 太多 | 结果明显错误 | RANSAC、鲁棒损失、匹配过滤 |
| 参数尺度差异大 | 收敛慢 | 归一化、合理单位 |
| 旋转参数化错误 | $R$ 不再正交 | 用 `rvec`、李代数或四元数 |
| 过拟合畸变 | 边缘看似更好，泛化变差 | 不随意增加高阶畸变参数 |
| 规范自由度未固定 | 解整体漂移 | 固定一个相机/尺度/坐标系 |

## 6.10 本章小结、问题与练习

最重要的三句话：

1. 优化的核心是定义残差，而不是盲目套算法。
2. 线性方法常用于初值，非线性优化用于最终精修。
3. RANSAC 和鲁棒损失是现实数据中不可缺少的防线。

### 基础篇小结

到这里，基础篇的数学链路已经闭合：

```text
相机模型 -> 外参 -> 平面单应/标定 -> 多视图 -> 深度表示 -> 优化
```
